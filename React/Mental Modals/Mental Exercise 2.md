## **🔹 Section 1: Core Philosophy**

1. What problem was React created to solve?
React was created to solve the difficulty of building and maintaining complex, dynamic user interfaces. As applications grew larger, manually updating the DOM became hard to manage and error-prone. React introduced a declarative and component-based approach where the UI is a function of state, allowing developers to build predictable, reusable, and efficiently updating interfaces.
